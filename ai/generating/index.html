Having Problems?

Stuck on the "generating" message (most common)

This can be caused by various reasons:

  • Ratelimits
    Every model has limits on how many requests it can send a day/hour.
    See this page for more detail.
  • Your account/IP has been locked.
    This can happen if you are sending too many requests or for other reasons.
  • Servers are down
    Always make sure to view our bot status.
  • Maintenance
    See announcements/downtime in our Discord.
